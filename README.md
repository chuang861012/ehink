# enink - A simple E-hentai CLI.
## Install
```
$ npm install -g elink
```
## Usage
### Command
```
$ elink
```
#### Entry
- E-hentai
- Exhentai **(cookies required)**
#### Keyboard control
`ESC`: Menu     
`s`: shortcut for search input
#### Favorites (cookies required)
Get favorites page of your account.
#### Popular (E-hentai only)
Popular page of E-hentai.
